## 👍서버 개발 과정👍
+ 먼저 기획의도에 따라 순서를 정함<br/>
1. **Serverless로 서버 배포**할 것<br/>
2. **Docker 활용** 하여 **패키징**할 것<br/>
3. **GitActions** 사용하여 **자동 배포** 할 것<br/>
4. 로그인 jwt<br/>
5. 콘서트 정렬 쿼리<br/>
6. 콘서트 상세보기 쿼리<br/>
7. 좋아요 leftJoin 쿼리<br/>
8. <br/>
